# Ansible 실행

```
$ ansible-playbook -i hosts get-user.yml

PLAY [호스트의 사용자 목록을 얻는다] ******************************************************************************************************************************************************************

TASK [Gathering Facts] *******************************************************************************************************************************************************************
ok: [127.0.0.1]

TASK [/etc/passwd에서 사용자 이름 추출] ***********************************************************************************************************************************************************
changed: [127.0.0.1]

TASK [명령을 실행한 결과에서 존재하는 사용자 이름을 목록으로 생성] *************************************************************************************************************************************************
ok: [127.0.0.1]

TASK [ansible 사용자가 있으면 메세지를 출력] **********************************************************************************************************************************************************
ok: [127.0.0.1] => {
    "msg": "이 서버에는 ansible 사용자가 존재합니다."
}

PLAY RECAP *******************************************************************************************************************************************************************************
127.0.0.1                  : ok=4    changed=1    unreachable=0    failed=0    skipped=0    rescued=0    ignored=0
```
